## How to contribute

### Did you find a bug?

* **Security vulnerabilities**: do not open a public issue. Contact the maintainers directly.
* Search existing [issues](https://github.com/redhat-sap/resource-agents.saphanasr-angi/issues) first.
* If no open issue exists, open a new one with a clear title, description, and as much relevant context as possible.

### Did you write a patch that fixes a bug?

* Open a pull request against the `main` branch.
* Describe the problem and solution clearly in the PR description. Include the relevant issue number if applicable.

### Do you intend to add a new feature or change an existing one?

* Open an issue first to discuss the change before writing code.
